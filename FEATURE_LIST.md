# Feature List

This file defines the product and engineering features this repository should track.

## Core MVP Features (Must Have)

1. Strict step-by-step typing mode with no skipping.
2. Correct/wrong key validation with blocked progression on mistakes.
3. Finger guidance for each target key (hand hint + keyboard key highlight).
4. Beginner lesson path (home row -> top row -> alphabet -> words -> sentences).
5. Slow-mode-first experience (progress and finger accuracy, no live WPM).
6. Repetition engine for weak keys during the same session.
7. Session summary with total chars, finger accuracy, and problematic finger guidance.
8. Progress persistence in `localStorage` (level unlock + last lesson + history).
9. Baseline accessibility (keyboard friendly, visible feedback text, contrast-safe UI).
10. Automated test coverage for core logic and critical user flows.

## Current Enhancements Included

1. Multilanguage support (English + Thai).
2. Refined UI hierarchy for focus typing.
3. Compact status chips and improved feedback animations.

## Phase 2 / Future Features

1. WPM and speed-oriented mode.
2. Custom text practice mode.
3. Advanced analytics (heatmaps, trends, deeper metrics).
4. Multiplayer/competitive features.
5. Word audio feedback in sentence mode (speak current word while typing).
